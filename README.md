## 🔗 Cara Install
<br>

- <code>git clone ↆↆ </code>
  
  ```bash
  git clone https://github.com/bagussatoto/Layanan-Voting.git
  ``` 
  
- <code>cd Layanan Voting</code>
- <code>npm install</code>
- <code>npm run dev</code>
